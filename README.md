# Arogya Gram

## Overview
Arogya Gram is a healthcare-focused project designed to improve healthcare accessibility and management for rural communities.

## Features
- Healthcare information management
- User-friendly interface
- Rural healthcare support system
- Digital accessibility for healthcare services

## Purpose
The project was developed to explore how technology can help improve healthcare accessibility and awareness in rural areas.

## Technologies Used
- HTML
- CSS
- JavaScript
- Database Concepts

## Status
Student project created for learning and practical implementation purposes.
